tokōのメニューはすべて愛情込めた手作り
お子様からお年寄りまで年齢を問わず、安心して召し上がっていただける幅広いメニューをご提供しています。


食材はtokō菜園で採れた野菜や新鮮な県内産野菜、卵は那須御養卵を使用。その日仕入れ立てのお魚、お肉、産地直送の新鮮な食材を使っています。旬の素材でその日のおすすめメニューもご用意しております。


大切な人と一緒に過ごす時間がもっとステキな時間になるように、お越しいただいた皆様の笑顔に寄り添う、あたたかいごはんを作っていきたい。tokōはそう想い続けています。
